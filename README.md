# Demo Assets

Simple repository for storing image/video assets for various demos.

## Usage

Reference assets using GitHub's raw URLs:


https://raw.githubusercontent.com/USERNAME/REPO/main/path/to/asset.jpg

## Structure

- `/images/` - Image files (PNG, JPG, GIF, etc.)
- `/videos/` - Video files (MP4, WebM, etc.)

## Contributing

Upload assets to the appropriate folder and reference them in your demos using the raw GitHub URLs.
